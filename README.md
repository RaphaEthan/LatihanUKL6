Fungsi
Program ini berfungsi sebagai kalkulator penjumlahan matriks. Program ini mengambil dua matriks (Matriks A dan Matriks B) dengan ukuran yang sama, lalu menjumlahkan elemen-elemen yang bersesuaian untuk menghasilkan matriks baru (Matriks C).

Cara Kerja
Program akan meminta pengguna memasukkan ukuran (dimensi) matriks terlebih dahulu:

"Masukkan jumlah baris matriks"

"Masukkan jumlah kolom matriks"

Program akan meminta pengguna memasukkan semua elemen (angka) untuk Matriks A satu per satu, sesuai dengan ukuran baris dan kolom.

Program akan meminta pengguna memasukkan semua elemen (angka) untuk Matriks B dengan cara yang sama.

Program melakukan penjumlahan matriks dengan menggunakan looping bersarang. Nilai Matriks C dihitung dengan rumus: C[i][j] = A[i][j] + B[i][j].

Setelah semua perhitungan selesai, program akan menampilkan "Hasil penjumlahan matriks (A + B):" dan mencetak seluruh elemen Matriks C ke layar.
